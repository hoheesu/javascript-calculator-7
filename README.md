# ☀️ javascript-calculator-precourse

## 🚀 문자열 덧셈 계산기

## 📋 기능 목록

- [x] 문자열 입력받기
- [x] 커스텀 구분자 지정하기 ➡️ ( `//`와 `\n` 사이에 위치한 문자가 있는 경우 )
- [ ] 쉼표`,`나 콜론`:`를 구분자로 지정하여 계산
- [ ] 구분자를 기준으로 숫자를 더한 값을 반환
- [ ] 문자열 입력 없을 시 0 반환
- [x] 잘못된 값 입력 시 예외처리 [ERROR] 반환이후 종료

---

### ⚠️ 예외처리 목록

- [x] 입력된 문자열이 커스텀 구분자나 숫자로 시작하지 않는경우
- [x] 커스텀 구분자의 형식이 잘못된
- [ ] 연속된 구분자가 입력된
